  D&Gstal - это приложение для обработки изображений с функцией невидимых водяных знаков на основе DCT (дискретного косинусного преобразования).

  Основной функционал:
- Режим редактирования изображений:
    > Регулировка яркости, контрастности и насыщенности с помощью слайдеров.
    > Crop-инструмент с указанием точных размеров.
    > Предпросмотр в реальном времени после каждого изменения.
    > Сброс фильтров к исходному состоянию.

- Встраивание невидимых цифровых водяных знакок
    > Встраивание текста в изображение с помощью DCT.
    > Регулировка силы водяного знака (0.01-1.0).
    > Стойкость к сжатию JPEG и базовой обработке.
    > Автоматическое восстановление водяного знака при проверке.

  Поддерживаемые форматы: JPEG/JPG, PNG, BMP.

  Использованные инструменты:
    > Python 3.8+ - основной язык программирования.
    > Tkinter - графический интерфейс пользователя.
    > Pillow (PIL) - обработка изображений.
    > NumPy - математические вычисления.
    > SciPy - DCT преобразования.
    > hashlib - генерация уникальных последовательностей.

  Установка и запуск:
  1. Клонирование репозитория
     bash
     git clone https://github.com/SofyZh/DnGstall_Zhuravleva
     cd dgstal
  2. Установка зависимостей
     bash
     pip install -r requirements.txt
  3. Запуск приложения
     bash
     python main.py
     
  Требования к библиотекам:
    Pillow>=9.0.0
    numpy>=1.21.0
    scipy>=1.7.0

  Основной рабочий процесс:
  1. Запуск приложения и основного меню.
  2. Выбор режима: "Edit Mode"\"Invisible Watermark"
  3. Выбор изображения для работы
  4.1. Режим редактирования: используйте слайды и crop-инструмент
  4.2. Режим водяных знаков: введите текст и настройте силу
  5. Сохранение результата в выбранном формате
